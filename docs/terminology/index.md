title: 术语
---

介绍计算机相关的术语知识

| 术语                  | 直译              | 修正                                                                                                                                                                                                                                          |
| --------------------- | ----------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| sprint                | 冲刺              | **迭代**,在计算机科学和软件开发领域，"sprint" 是一个常用术语，尤其在敏捷开发（Agile Development）中。它指的是一个较短的、时间固定的开发周期，通常为一到四周，用于完成特定的一组任务或功能。所以翻译为迭代更合适，在这用"冲刺"显得有点无厘头。 |
| blockchain            | 区块链            | **区块链**，一种分布式数据存储技术，通过密码学原理将数据块按时间顺序连接成链状结构，能确保数据不可篡改且可追溯。                                                                                                                              |
| smart contract        | 智能合约          | **智能合约**，在区块链上运行的自动执行的程序，当预设条件满足时自动执行合约条款，无需第三方干预。                                                                                                                                              |
| token                 | 代币              | **代币**，在区块链网络中代表某种资产或使用权的数字单位。包括加密货币和非同质化代币(NFT)等。                                                                                                                                                   |
| wallet                | 钱包              | **钱包**，在Web3中指存储和管理用户加密资产的软件或硬件设备，包含私钥用于签名交易。                                                                                                                                                            |
| dApp                  | 去中心化应用      | **去中心化应用**，建立在区块链上的应用程序，不受单一实体控制，通常通过智能合约实现其核心功能。                                                                                                                                                |
| mining                | 挖矿              | **挖矿**，通过提供计算能力来验证和处理区块链交易的过程，成功的矿工会获得相应的代币奖励。                                                                                                                                                      |
| gas fee               | 燃料费            | **交易手续费**，在以太坊等区块链上执行交易或智能合约所需支付的费用，用于补偿验证者处理交易的计算资源。                                                                                                                                        |
| staking               | 质押              | **质押**，将加密货币锁定作为抵押品以参与网络验证过程，获取奖励的机制。                                                                                                                                                                        |
| Web3                  | Web3              | **Web3**，互联网的新范式，强调去中心化、区块链技术和代币经济，用户拥有自己的数据和数字资产。                                                                                                                                                  |
| frontend              | 前端              | **前端**，应用程序中直接与用户交互的部分，负责呈现界面和处理用户输入，通常使用HTML、CSS和JavaScript等技术构建。                                                                                                                               |
| backend               | 后端              | **后端**，应用程序中处理业务逻辑、数据存储和服务器操作的部分，对用户不可见但支撑整个应用运行。                                                                                                                                                |
| API                   | 应用程序接口      | **应用程序接口**，允许不同软件组件相互通信的规则和工具集，在Web开发中常用于前后端数据交换。                                                                                                                                                   |
| framework             | 框架              | **框架**，提供基础结构和功能的软件库集合，开发者可以在其上构建应用，如前端的React、Vue，后端的Express、Django等。                                                                                                                             |
| responsive design     | 响应式设计        | **响应式设计**，使网页能够自适应不同设备屏幕尺寸的设计方法，确保在手机、平板和电脑上都有良好的显示效果。                                                                                                                                      |
| dependency            | 依赖              | **依赖**，项目所需的外部代码库或模块，通常通过包管理器(如npm、pip)进行管理。                                                                                                                                                                  |
| server-side rendering | 服务器端渲染      | **服务器端渲染(SSR)**，在服务器上生成完整HTML页面后发送给客户端的技术，可提高首屏加载速度和搜索引擎优化。                                                                                                                                     |
| middleware            | 中间件            | **中间件**，在操作系统或应用程序的不同组件之间传递数据的软件，在Web开发中常用于处理HTTP请求和响应的中间逻辑。                                                                                                                                 |
| deployment            | 部署              | **部署**，将开发完成的应用程序发布到生产环境的过程，使其可被最终用户访问和使用。                                                                                                                                                              |
| scaling               | 扩展              | **扩展**，增加系统处理能力以适应更多用户或数据的过程，包括垂直扩展(增强单个节点)和水平扩展(增加节点数量)。                                                                                                                                    |
| consensus mechanism   | 共识机制          | **共识机制**，区块链网络中用于验证交易并达成网络一致性的算法，常见的有工作量证明(PoW)和权益证明(PoS)等。                                                                                                                                      |
| sharding              | 分片              | **分片**，将区块链网络分割成多个小部分(分片)的扩展方案，每个分片可以并行处理交易，提高整体网络吞吐量。                                                                                                                                        |
| oracle                | 预言机            | **预言机**，连接区块链与外部世界的服务，将现实世界的数据安全地引入智能合约中，使合约能够基于外部信息执行。                                                                                                                                    |
| layer 2               | 第二层            | **第二层解决方案**，建立在基础区块链(第一层)之上的扩展技术，旨在提高交易速度和降低费用，如比特币的闪电网络和以太坊的Optimistic Rollups。                                                                                                      |
| NFT                   | 非同质化代币      | **非同质化代币**，区块链上独特且不可互换的数字资产，每个代币都有唯一标识符和元数据，常用于数字艺术品、收藏品和虚拟资产。                                                                                                                      |
| DeFi                  | 去中心化金融      | **去中心化金融**，建立在区块链上的金融服务生态系统，无需传统金融中介，包括借贷、交易和资产管理等功能。                                                                                                                                        |
| DAO                   | 去中心化自治组织  | **去中心化自治组织**，基于区块链的组织形式，通过智能合约和社区投票实现自治管理，没有传统的层级结构。                                                                                                                                          |
| cold wallet           | 冷钱包            | **冷钱包**，离线存储加密资产的钱包，不连接互联网，提供较高安全性，常见形式有硬件钱包和纸钱包。                                                                                                                                                |
| hot wallet            | 热钱包            | **热钱包**，连接互联网的加密资产钱包，便于使用但安全性相对较低，包括手机钱包和网页钱包等。                                                                                                                                                    |
| metaverse             | 元宇宙            | **元宇宙**，集成了虚拟现实、增强现实和互联网的沉浸式数字环境，用户可在其中交互、社交和参与各种活动。                                                                                                                                          |
| liquidity pool        | 流动性池          | **流动性池**，去中心化交易所中用户提供的代币资金池，促进自动化交易并为流动性提供者产生收益。                                                                                                                                                  |
| yield farming         | 收益耕作          | **收益耕作**，在DeFi平台上通过提供资产流动性或参与各种协议来获取额外代币奖励的策略。                                                                                                                                                          |
| web component         | Web组件           | **Web组件**，一组网页开发标准，允许创建可重用、封装的自定义HTML元素，包括自定义元素、Shadow DOM和HTML模板。                                                                                                                                   |
| state management      | 状态管理          | **状态管理**，在前端应用中管理和维护组件间共享数据的技术或工具，如Redux、Vuex等，帮助处理复杂应用的数据流。                                                                                                                                   |
| bundler               | 打包工具          | **打包工具**，将多个JavaScript模块及其依赖合并为适合浏览器使用的优化包的工具，如Webpack、Rollup和Parcel等。                                                                                                                                   |
| transpiler            | 转译器            | **转译器**，将一种编程语言的代码转换为另一种语言或版本的工具，如Babel可将现代JavaScript转换为兼容旧浏览器的代码。                                                                                                                             |
| CDN                   | 内容分发网络      | **内容分发网络**，分布在全球各地的服务器网络，用于快速分发网站内容，减少加载时间并提高可用性。                                                                                                                                                |
| CI/CD                 | 持续集成/持续部署 | **持续集成/持续部署**，自动化软件开发流程，持续集成自动测试代码变更，持续部署自动将验证后的代码发布到生产环境。                                                                                                                               |
| microservices         | 微服务            | **微服务**，将应用程序设计为一系列松耦合、可独立部署的小型服务的架构风格，每个服务负责特定功能并可独立扩展。                                                                                                                                  |
| REST API              | REST API          | **表现层状态转移API**，一种Web服务设计风格，使用HTTP方法和标准状态码，强调资源的无状态操作和统一接口。                                                                                                                                        |
| GraphQL               | GraphQL           | **GraphQL**，一种用于API的查询语言和运行时，允许客户端精确指定所需数据，减少网络请求并避免过度获取或获取不足问题。                                                                                                                            |
| WebSockets            | WebSockets        | **WebSockets**，提供浏览器和服务器之间全双工通信的协议，支持实时数据传输，不需要反复建立HTTP连接。                                                                                                                                            |
| ORM                   | 对象关系映射      | **对象关系映射**，将数据库中的数据转换为面向对象编程语言中的对象的技术，简化数据库操作并提供数据抽象层。                                                                                                                                      |
| serverless            | 无服务器          | **无服务器**，一种云计算执行模型，开发者无需管理服务器，只专注于代码，云提供商负责基础设施管理和按需扩展。                                                                                                                                    |
| authentication        | 身份验证          | **身份验证**，验证用户身份的过程，确保只有授权用户才能访问系统，常见方法包括密码、多因素认证和OAuth等。                                                                                                                                       |
| authorization         | 授权              | **授权**，确定已验证身份的用户可以访问哪些资源或执行哪些操作的过程，通常通过角色或权限系统实现。                                                                                                                                              |
| virtualization        | 虚拟化            | **虚拟化**，创建计算资源虚拟版本的技术，如虚拟机或容器，使多个系统可以在同一物理硬件上运行，提高资源利用率。                                                                                                                                  |
| containerization      | 容器化            | **容器化**，将应用程序及其依赖打包到一个容器中的技术，确保在不同环境中一致运行，Docker是最流行的容器平台。                                                                                                                                    |
| load balancing        | 负载均衡          | **负载均衡**，分配网络流量到多个服务器的技术，以优化资源使用、最大化吞吐量、减少延迟并确保高可用性。                                                                                                                                          |
| Solidity              | Solidity          | **Solidity**，以太坊区块链上开发智能合约的主要编程语言，面向对象，语法类似JavaScript，专为实现智能合约功能而设计。                                                                                                                            |
| EVM                   | 以太坊虚拟机      | **以太坊虚拟机**，以太坊网络中执行智能合约的运行环境，负责处理交易并在区块链上维护合约状态。                                                                                                                                                  |
| gas limit             | 燃料上限          | **燃料上限**，以太坊交易中用户愿意为交易执行支付的最大计算单位，防止无限循环和资源耗尽攻击。                                                                                                                                                  |
| gas price             | 燃料价格          | **燃料价格**，用户愿意为每单位燃料支付的价格，通常以Gwei（以太币的一个小单位）计算，影响交易处理优先级。                                                                                                                                      |
| wei                   | 微                | **wei**，以太币的最小单位，1以太币等于10^18 wei，类似于比特币中的聪。                                                                                                                                                                         |
| private key           | 私钥              | **私钥**，用于签署交易的秘密密钥，控制加密资产的所有权证明，必须妥善保管，一旦丢失无法恢复。                                                                                                                                                  |
| public key            | 公钥              | **公钥**，从私钥派生的密钥，用于验证交易签名，公钥派生的地址可安全地公开用于接收资产。                                                                                                                                                        |
| mnemonic phrase       | 助记词            | **助记词**，一系列单词，用于生成和恢复加密钱包的私钥，通常为12或24个单词，需安全保存。                                                                                                                                                        |
| ABI                   | 应用二进制接口    | **应用二进制接口**，在Solidity中定义智能合约函数和参数的JSON格式接口描述，前端应用用它与区块链上的合约交互。                                                                                                                                  |
| bytecode              | 字节码            | **字节码**，Solidity合约编译后的低级代码，部署到区块链时执行的实际代码，EVM可以解释和执行。                                                                                                                                                   |
| remix                 | Remix             | **Remix**，以太坊官方提供的基于浏览器的IDE，用于编写、测试和部署智能合约。                                                                                                                                                                    |
| truffle               | Truffle           | **Truffle**，流行的智能合约开发框架，提供编译、链接、部署和测试等功能的自动化工具。                                                                                                                                                           |
| hardhat               | Hardhat           | **Hardhat**，以太坊开发环境，提供本地开发网络、测试和调试工具，适合专业智能合约开发。                                                                                                                                                         |
| metamask              | MetaMask          | **MetaMask**，浏览器扩展钱包，连接普通网页浏览器和以太坊区块链，允许用户与dApp交互。                                                                                                                                                          |
| web3.js               | Web3.js           | **Web3.js**，以太坊JavaScript API，允许前端应用与以太坊区块链交互，发送交易和调用智能合约。                                                                                                                                                   |
| ethers.js             | Ethers.js         | **Ethers.js**，轻量级JavaScript库，用于与以太坊区块链交互，提供钱包功能、合约调用和交易管理。                                                                                                                                                 |
| IPFS                  | 星际文件系统      | **星际文件系统**，去中心化的文件存储协议，常用于存储NFT元数据和dApp内容，通过内容寻址而非位置寻址实现。                                                                                                                                       |
| Node.js               | Node.js           | **Node.js**，基于Chrome V8引擎的JavaScript运行时环境，允许在服务器端运行JavaScript代码，处理后端逻辑。                                                                                                                                        |
| npm                   | npm包管理器       | **npm包管理器**，Node.js默认的包管理工具，用于安装、共享和管理JavaScript代码包和依赖。                                                                                                                                                        |
| yarn                  | Yarn              | **Yarn**，Facebook开发的JavaScript包管理器，是npm的替代品，提供更快的安装速度和更可靠的依赖管理。                                                                                                                                             |
| Express               | Express           | **Express**，流行的Node.js Web应用框架，提供简洁API用于构建Web服务器和API，支持中间件架构。                                                                                                                                                   |
| Koa                   | Koa               | **Koa**，由Express团队开发的新一代Node.js Web框架，更小巧、表达力更强，使用异步函数处理中间件。                                                                                                                                               |
| Nest.js               | Nest.js           | **Nest.js**，基于TypeScript构建的Node.js服务器端框架，结合OOP、FP和FRP元素，适合构建可扩展的企业级应用。                                                                                                                                      |
| MongoDB               | MongoDB           | **MongoDB**，流行的NoSQL数据库，使用JSON格式存储文档数据，非关系型设计使其适合处理大量非结构化数据。                                                                                                                                          |
| Mongoose              | Mongoose          | **Mongoose**，MongoDB对象模型工具，提供模式验证、关系模型和中间件支持，简化Node.js应用中的MongoDB使用。                                                                                                                                       |
| JSON Web Token        | JSON Web令牌      | **JSON Web令牌(JWT)**，紧凑的自包含数据传输格式，用于在服务和客户端之间安全传输信息，常用于身份验证和授权。                                                                                                                                   |
| Socket.IO             | Socket.IO         | **Socket.IO**，JavaScript库，支持服务器和客户端之间的实时双向通信，在WebSockets不可用时自动降级到其他传输方式。                                                                                                                               |
| webpack               | Webpack           | **Webpack**，现代JavaScript应用的静态模块打包工具，处理应用依赖并生成优化的静态资源包。                                                                                                                                                       |
| Babel                 | Babel             | **Babel**，JavaScript编译器，将最新版本的JavaScript代码转换为向后兼容的版本，使开发者能使用新语言特性。                                                                                                                                       |
| ESLint                | ESLint            | **ESLint**，JavaScript代码静态分析工具，帮助开发者发现并修复代码问题，强制执行一致的编码风格。                                                                                                                                                |
| React                 | React             | **React**，Facebook开发的JavaScript库，用于构建用户界面，特别是单页应用，基于组件的架构促进代码复用。                                                                                                                                         |
| Vue                   | Vue               | **Vue**，渐进式JavaScript框架，用于构建用户界面，易于上手且集成方便，支持响应式数据绑定和组件化开发。                                                                                                                                         |
| Angular               | Angular           | **Angular**，Google维护的全功能前端框架，提供完整MVC架构，包含路由、表单处理和依赖注入等功能。                                                                                                                                                |
| JSX                   | JSX               | **JSX**，JavaScript的语法扩展，允许在JavaScript中编写类似HTML的代码，常用于React组件的开发。                                                                                                                                                  |
| hooks                 | 钩子              | **钩子**，React 16.8引入的特性，允许函数组件使用状态和其他React特性，如useState和useEffect等。                                                                                                                                                |
| virtual DOM           | 虚拟DOM           | **虚拟DOM**，内存中的DOM表示，用于优化UI更新性能，通过比较虚拟DOM的差异来最小化对实际DOM的操作。                                                                                                                                              |
| event loop            | 事件循环          | **事件循环**，JavaScript的执行模型，处理异步操作的机制，确保非阻塞操作的执行顺序和回调处理。                                                                                                                                                  |
| callback              | 回调              | **回调**，传递给另一个函数的函数，在特定事件或条件满足时执行，是JavaScript处理异步操作的基本方式。                                                                                                                                            |
| promise               | 承诺/期约         | **承诺/期约**，表示异步操作最终完成或失败的对象，提供比回调更清晰的方式处理异步代码。                                                                                                                                                         |
| async/await           | 异步/等待         | **异步/等待**，处理Promise的语法糖，使异步代码看起来和执行起来更像同步代码，提高可读性和可维护性。                                                                                                                                            |
| 51% attack            | 51%攻击           | **51%攻击**，当单个实体控制区块链网络超过50%的算力或质押量时的安全风险，可能导致交易回滚和双重支付。                                                                                                                                          |
| airdrop               | 空投              | **空投**，加密项目向社区成员免费分发代币的营销策略，通常要求用户完成特定任务或满足特定条件。                                                                                                                                                  |
| atomic swap           | 原子交换          | **原子交换**，允许不同区块链上的代币直接交换的技术，无需中心化交易所，完全在链上完成且保证安全。                                                                                                                                              |
| bridge                | 桥接              | **桥接**，连接不同区块链网络的协议，允许资产和信息在不同区块链间转移，解决跨链互操作性问题。                                                                                                                                                  |
| burn                  | 销毁              | **销毁**，永久性地从流通供应中移除代币的过程，通常通过发送到无法访问的地址实现，减少总供应量。                                                                                                                                                |
| testnet               | 测试网            | **测试网**，区块链的测试版本，允许开发者在不使用实际价值代币的环境中测试应用和合约。                                                                                                                                                          |
| mainnet               | 主网              | **主网**，区块链的生产版本，处理实际价值交易，与测试网相对，上线应用和合约的最终环境。                                                                                                                                                        |
