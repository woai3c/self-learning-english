介绍计算机相关的术语知识

| 术语   | 直译 | 修正                                                                                                                                                                                                                                          |
| ------ | ---- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| sprint | 冲刺 | **迭代**,在计算机科学和软件开发领域，"sprint" 是一个常用术语，尤其在敏捷开发（Agile Development）中。它指的是一个较短的、时间固定的开发周期，通常为一到四周，用于完成特定的一组任务或功能。所以翻译为迭代更合适，在这用“冲刺”显得有点无厘头。 |
