# 现在进行时 present continuous

这种时态用于描述正在进行的动作或状态，或者在未来有计划或安排的动作。

现在进行时的构成是：主语 + am/is/are + 动词的现在分词（即动词+ing）。

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;I'm doing

past----**now**----future

记住 `I am doing` 的格式，中间有谓词连接。

## am/is/are (not) + -ing

- I'**m** work**ing**.
- She **isn't** eat**ing**.
- We'**re** hav**ing** dinner.
